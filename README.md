# 🌤️ WeatherApp

A simple, elegant weather application that provides real-time weather information for any city worldwide.

## Features

- 🔍 **City Search** - Search weather by city name
- 🌡️ **Temperature Display** - Shows current temperature in Celsius
- 📊 **Weather Details** - Displays weather condition, high/low temperatures
- 🎨 **Beautiful UI** - Modern gradient design with smooth user experience
- 🌐 **Real-time Data** - Powered by OpenWeatherMap API

## Live Demo

Check out the live application: [Weather App](https://weather-app-lyart-eight-42.vercel.app)

## Tech Stack

- **Frontend:** HTML5, CSS3, JavaScript (Vanilla)
- **Weather API:** [OpenWeatherMap API](https://openweathermap.org/api)
- **Icons:** Font Awesome 6.5.0
- **Deployment:** Vercel

## Project Structure

```
WeatherApp/
├── index.html       # Main HTML structure
├── style.css        # Styling and layout
└── app.js           # JavaScript logic and API calls
```

## How to Use

1. Clone the repository
2. Open `index.html` in your browser
3. Enter a city name in the search bar
4. Click the search icon to see weather details
5. View temperature, conditions, and highs/lows

## How It Works

The app fetches real-time weather data from the OpenWeatherMap API and displays:
- Current temperature
- Weather conditions with icon
- High and low temperatures
- Current date

## Installation

No installation required! Simply open the HTML file in any modern web browser.
